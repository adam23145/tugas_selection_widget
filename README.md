# tugas_selection_widget

Selection Widget berfungsi untuk membuat sebuah daftar /list yang berisi banyak pilihan atau item.
Terdapat beberapa Selection Widget, antara lain:

1. ListView
2. Spinner
3. AutoCompleteTextView
4. Recycle View
